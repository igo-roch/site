# Wedding Invitation

Статическая HTML-верстка по макету Figma лежит в `index.html`.

Для локального просмотра достаточно открыть `D:\Users\iIgor\Documents\Plugin\renamer\wedding-invitation\index.html` в браузере.

Что важно:

- Все изображения сохранены локально в папке `assets`, поэтому страница не зависит от временных Figma URL.
- Для точного совпадения заголовка нужен исходный файл шрифта `Martina scriptC` в `woff2` или `woff`. Сейчас для него стоит fallback.
- Остальные шрифты подтягиваются через Google Fonts.
